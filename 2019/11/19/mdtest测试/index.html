Mdtest参数说明

选项 说明
-F 只创建文件
-D 只针对目录进行测试
-L 只在目录树的子目录层创建文件/目录
-z 目录树深度（从根到叶子节点的深度）
-b 目录树的分支数（非叶子节点的分支）
-I 每个节点下的目录或文件数
-u 为每个工作任务指定工作目录
-d 指出测试运行的目录
-i c测试循环的次数
-n 每个进程将要创建/统计/删除的目录和文件数目
-w 文件创建以后写入多少bytes
-p 循环前的时间延迟
